The Essentials

  • Mid-2019 15" Macbook Pro

    As much as I don't love the keyboard on this laptop (the 2019 16" model is much better), it gets the job done, and I've been very happy with it otherwise.

  • iPad Pro 11"

    Beautiful tablet, great for taking notes in meetings or watching movies while travelling.

  • Google Pixel 4 XL

    I've owned every major generation of the Pixel, I love this phone.

  • Bose NC 700

    Possibly the best QOL purchase I've made as a remote worker, noise cancelling headphones allow me to work anywhere in isolation. I've also owned the Sony XM2 and XM3, which had slightly better noise cancelling, but the Bose NC 700 has the ability to connect to two devices simultaneously over Bluetooth, which is pretty cool, and very convenient.

Software

  • Brave

    I recommend Brave to everyone, it lives up to its promises.

  • Sublime Text

    I keep planning on switching to VS Code, but I really like the simplicity of Sublime. :shrug:

  • Insomnia

    Fantastic API testing client, encourages you to work in "environments", has been an easy switch from Postman

  • Front

    I use Front for all of my business-focused email. It makes collaboration super easy, and it's a breeze to set up forwarding addresses for new web properties.

  • Hyper
  • Dashlane
